- **After every task commit:** Run `cargo test -p <touched-crate> --lib` (scoped quick run)
- **After every plan wave:** Run `cargo test --workspace` + `cargo clippy --workspace --all-targets -- -D warnings`
- **Before `/gsd:verify-work`:** Full suite must be green + live integration test per CLAUDE.md
- **Max feedback latency:** 180 seconds
