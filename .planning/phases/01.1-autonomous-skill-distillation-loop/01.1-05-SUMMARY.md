## Decisions Made

- **TOML template for distilled skills** uses `[runtime] type = "promptonly"` + `[[tools.provided]]` — minimum valid manifest, matches the `skill_toml` helper used in registry tests. The plan's original template was missing `[runtime]` and `[[tools.provided]]` which would have failed `toml::from_str::<SkillManifest>`.
- **Description quote-escaping**: `replace('"', "'")` before embedding the LLM answer as TOML string literal — prevents parse failures on LLM output containing quotes.
- **distillation_cap_state pre-captured**: `home_dir` is accessed before `config` is moved into the struct literal. Required local variable capture before `let kernel = Self { config, ... }`.
- **`std::io::Error::other(e)` used** instead of `std::io::Error::new(std::io::ErrorKind::Other, e)` — clippy lint `io_other_error` caught and fixed inline (Rule 1: auto-fix).
